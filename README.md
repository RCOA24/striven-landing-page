# Striven Landing Page

A modern, high-performance landing page for **Striven**, the AI-powered fitness companion. This project showcases a sleek, professional design with "Apple-like" aesthetics, smooth animations, and a fully responsive layout.

![Striven Logo](/public/StrivenLogo.png)

##  Features

-   **Immersive Hero Section**: Features a dynamic background, glassmorphism effects, and a realistic 3D phone mockup with a video placeholder.
-   **Interactive Features Grid**: A 4x2 grid highlighting 8 core features of the Striven app, including AI Food Scanner, Live Step Map, and Data Management.
-   **Infinite Tech Stack Carousel**: A smooth, horizontal scrolling marquee displaying the modern technologies powering the app.
-   **Dark & Light Mode**: Fully supported theme toggling with persistent state.
-   **Responsive Design**: Optimized for mobile, tablet, and desktop devices.
-   **PWA Ready**: Direct links to launch the Progressive Web App.

##  Tech Stack

Built with the latest web technologies for speed and performance:

-   **Framework**: [React 19](https://react.dev/)
-   **Build Tool**: [Vite 7](https://vitejs.dev/)
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
-   **Animations**: [Framer Motion](https://www.framer.com/motion/)
-   **Icons**: [Lucide React](https://lucide.dev/)

-   **Unified Leaderboard**: Cross-platform rank syncing.
